# plugins/enver-framework/commands - içindekiler

> Bu belge otomatik üretilir. Elle satır eklenmez;
> açıklamalar dosyaların kendisinden okunur.

## Dosyalar

| Dosya | Ne işe yarar |
|-------|--------------|
| `ara.md` | Tek arama noktasi - hafiza, bilgi notlari, proje tanimlari ve icindekiler belgelerinde birlikte arar |
| `backup.md` | Veritabani ve dosya yedekleme - pg_dump, restore, uploads yedek, otomatik yedek |
| `canli-kontrol.md` | Localhost tarayicide acip kontrol eder - gelistirme ortaminda UI ve fonksiyon testi |
| `db-yonetimi.md` | Veritabani yonetimi - migration, seed, tablo incele, sorgu calistir |
| `dokumantasyon.md` | Dokumantasyon olustur - API endpoint listesi, DB sema, changelog, mimari diyagram |
| `durum-kaydet.md` | Conversation handoff - nerede kaldigini kaydeder, sonraki konusmaya hazirlar |
| `faz-kontrol.md` | Aktif fazin kapi kontrollerini calistirir - lint, type check, guvenlik, test, build |
| `faz.md` | Faz motoru - aktif faz, ilerleme, kapi kontrolu, calisma modu ve tam yetki |
| `framework-ayarlari.md` | Framework ayarlari - memory, vault, hook, izin, versiyon yonetimi |
| `git-islemleri.md` | Git islemleri - branch, commit, PR, merge, conflict cozme |
| `guvenlik-tara.md` | Projedeki guvenlik aciklarini tarar - OWASP Top 10, credentials sizintisi, bagimlilik kontrolu |
| `hafiza.md` | Hafiza - nerede kaldik, karar defteri, hata kutuphanesi, oturum ozetleri ve proje index'leri |
| `index.md` | Komut rehberi - butun komutlar, beceriler, ajanlar ve korumalar tek listede, detayli kullanimla |
| `kasa.md` | Parola korumali kasa - sifre ve anahtarlari acar, okur, gunceller, kilitler |
| `kesif.md` | Kodlamadan onceki dort asama - istek toplama, arastirma, netlestirme, plan |
| `log-izle.md` | PM2, Nginx loglarini izle - hata filtrele, arama yap, canli izle |
| `monitoring.md` | Sunucu sagligi, API uptime, SSL sertifika, CPU/RAM/disk, PM2 ve DB baglanti kontrolu |
| `panel.md` | Framework ana menusu - tum komutlara tek noktadan eris, 4 sekme 16 kategori 80+ islem |
| `proje-baslat.md` | Yeni proje sablonla baslatir - CLAUDE.md, faz plani, dizin yapisi olusturur |
| `projeler.md` | Butun projeler tek ekranda - durum, musteri, gorev; projeler arasi gecis ve gecmeden sorgu |
| `saglik.md` | Framework saglik kontrolu - korumalar gercekten calisiyor mu, cakisma var mi, paylasima hazir mi |
| `sema.md` | Gorsel sistem semasi - hangi projede ne calisiyor, ne amacla, hangi projeye bagli |
| `senkron.md` | Coklu bilgisayar senkronu - makine tanitma, hafizayi cekme ve gonderme, cakisma kontrolu |
| `tasarim.md` | Ozgun tasarim - kimlik uretimi, ilham kaynaklari, kalip denetimi ve iz imzasi |
| `temizlik.md` | Memory, plan ve gecici dosyalari temizler - duplikatlari siler, arsivler |
| `toplu-islemler.md` | Toplu islemler - ESLint auto-fix, import temizle, TODO listele, paket guncelle, dead code bul |
| `web-kontrol.md` | Canli siteyi tarayicide acip kontrol eder - UI, fonksiyon, giris, test, link tarama |

---

Açıklaması eksik olan bir dosya varsa iki yol var:
dosyanın başına açıklama yaz, ya da bu klasöre `aciklamalar.json` koy.
